================================================================================
🔓 COMO ACESSAR A PÁGINA DE CONFIGURAÇÕES DO SISTEMA
================================================================================

✅ O QUE FOI FEITO:

1. ✅ Página "Configurações do Sistema" adicionada ao menu
2. ✅ Menu aparece na seção Administrativa (após Permissões)
3. ✅ Scripts criados para adicionar permissão no banco de dados

================================================================================
⚡ SOLUÇÃO RÁPIDA (SIGA ESTES PASSOS):
================================================================================

PASSO 1: Adicionar a Permissão
------------------------------
Acesse no navegador:
http://localhost/hol/database/adicionar_permissao_configuracoes.php

Isso vai criar a permissão e adicionar ao grupo Administrador automaticamente.

PASSO 2: Fazer Logout e Login
------------------------------
IMPORTANTE! Faça logout do sistema e faça login novamente para as permissões
serem atualizadas.

PASSO 3: Verificar o Menu
--------------------------
Olhe no menu lateral, na seção administrativa, você deve ver:

    👥 Usuários
    📊 Grupos  
    🔒 Permissões
    ⚙️ Configurações do Sistema  ← NOVO!

PASSO 4: Acessar a Página
--------------------------
Clique em "Configurações do Sistema" ou acesse diretamente:
http://localhost/hol/configuracoes_sistema.php

================================================================================
❓ PROBLEMAS?
================================================================================

Se não aparecer no menu:
-------------------------
1. Execute: http://localhost/hol/database/adicionar_permissao_configuracoes.php
2. Faça LOGOUT
3. Faça LOGIN novamente
4. Limpe o cache (Ctrl + F5)

Se der "Acesso Negado":
-----------------------
1. Verifique se você é Administrador
2. Execute o script de permissão novamente
3. Faça logout e login
